## Description

On switch-in, this Pokemon is alerted if any opposing Pokemon has an attacking move with a type that is super effective against this Pokemon, or any OHKO move. This effect considers Hidden Power to be its determined type, and every other move to be its original type.

## Mechanics on Pokéfind

Functions on Pokéfind as described above. Implemented in the battle simulator.
