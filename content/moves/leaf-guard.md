## Description

If Sunny Day is active, this Pokemon cannot become affected by a non-volatile status condition or Yawn, and Rest will fail for it. This effect is prevented if this Pokemon is holding a Utility Umbrella.

## Mechanics on Pokéfind

Listed in the Pokéfind ability registry but currently has **no in-battle effect** — the battle simulator does not implement the trigger. Treat as flavor only until the behaviour is added.
