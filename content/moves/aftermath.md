## Description

If this Pokemon is knocked out with a contact move, that move's user loses 1/4 of its maximum HP, rounded down. This effect is prevented if the move's user has the Magic Guard Ability or if any active Pokemon has the Damp Ability.

## Mechanics on Pokéfind

Functions on Pokéfind as described above. Implemented in the battle simulator.
