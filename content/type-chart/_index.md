Three modes share this page:

- **Type vs. Type** — pick an attacker type and one or two defender types; the multiplier updates live.
- **Defense Profile** — pick any Pokémon by name and see all 18 attacking types graded against its typing at once.
- **Full Chart** — the classic 18×18 reference grid for single-type defenders.

Type matchups follow the Gen 6+ chart (the same one the Damage Calc uses).
