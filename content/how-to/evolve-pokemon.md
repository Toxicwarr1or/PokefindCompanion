A practical reference for how Pokémon evolve on Pokéfind. There's no single "evolve" command — the game watches for the right condition and triggers the evolution automatically. Most often, that condition is **level up**, but several Pokémon need something more specific.

> For the **items themselves** (which stone evolves who, which item is held during which trade, where to get them), see **[Evolution Items](/evolution-items/)**.

---

## 1. Level up

The default. The Pokémon evolves the next time it gains a level past its evolution threshold.

- **Examples:** [Charmander](/pokedex/charmander/) (16) → [Charmeleon](/pokedex/charmeleon/) (36) → [Charizard](/pokedex/charizard/), [Magikarp](/pokedex/magikarp/) (20) → [Gyarados](/pokedex/gyarados/), [Patrat](/pokedex/patrat/) (20) → [Watchog](/pokedex/watchog/).

If a Pokémon should have evolved by now and hasn't, check whether it's **holding an [Everstone](/evolution-items/)** — that blocks every evolution path.

---

## 2. Level up at a specific time of day

Some Pokémon only evolve when they level up during the **day** or **night** in-game. The clock is the world's daylight cycle, not real time.

- **Day:** [Eevee](/pokedex/eevee/) (high friendship + day) → [Espeon](/pokedex/espeon/). [Happiny](/pokedex/happiny/) + [Oval Stone](/evolution-items/#held--level-up-no-trade) + day → [Chansey](/pokedex/chansey/).
- **Night:** [Eevee](/pokedex/eevee/) (high friendship + night) → [Umbreon](/pokedex/umbreon/). [Sneasel](/pokedex/sneasel/) + [Razor Claw](/evolution-items/#held--level-up-no-trade) + night → [Weavile](/pokedex/weavile/). [Gligar](/pokedex/gligar/) + [Razor Fang](/evolution-items/#held--level-up-no-trade) + night → [Gliscor](/pokedex/gliscor/).

---

## 3. Friendship / happiness

Several Pokémon evolve when they level up while their **friendship** is high. Friendship goes up from being in your party, gaining levels, using vitamins, and not fainting often.

- **Pure friendship:** [Pichu](/pokedex/pichu/) → [Pikachu](/pokedex/pikachu/). [Cleffa](/pokedex/cleffa/) → [Clefairy](/pokedex/clefairy/). [Igglybuff](/pokedex/igglybuff/) → [Jigglypuff](/pokedex/jigglypuff/). [Golbat](/pokedex/golbat/) → [Crobat](/pokedex/crobat/). [Chansey](/pokedex/chansey/) → [Blissey](/pokedex/blissey/). [Munchlax](/pokedex/munchlax/) → [Snorlax](/pokedex/snorlax/). [Riolu](/pokedex/riolu/) → [Lucario](/pokedex/lucario/).
- **Friendship + day:** [Eevee](/pokedex/eevee/) → [Espeon](/pokedex/espeon/).
- **Friendship + night:** [Eevee](/pokedex/eevee/) → [Umbreon](/pokedex/umbreon/).

---

## 4. Use an evolution stone

Apply the stone directly to a compatible Pokémon. See **[Evolution Items](/evolution-items/#stones-use-on-pokémon)** for the full Pokémon-per-stone table.

- **Examples:** [Vulpix](/pokedex/vulpix/) + [Fire Stone](/evolution-items/#stones-use-on-pokémon) → [Ninetales](/pokedex/ninetales/). [Pikachu](/pokedex/pikachu/) + [Thunder Stone](/evolution-items/#stones-use-on-pokémon) → [Raichu](/pokedex/raichu/). [Eevee](/pokedex/eevee/) + [Water Stone](/evolution-items/#stones-use-on-pokémon) → [Vaporeon](/pokedex/vaporeon/). [Snorunt](/pokedex/snorunt/) (female) + [Dawn Stone](/evolution-items/#stones-use-on-pokémon) → [Froslass](/pokedex/froslass/). [Kirlia](/pokedex/kirlia/) (male) + [Dawn Stone](/evolution-items/#stones-use-on-pokémon) → [Gallade](/pokedex/gallade/).

---

## 5. Trade

Hand the Pokémon off to another player. The evolution fires on the receive side. Pokéfind handles trade evolutions the standard way — *they do require an actual trade*.

- **Trade alone:** [Kadabra](/pokedex/kadabra/) → [Alakazam](/pokedex/alakazam/). [Machoke](/pokedex/machoke/) → [Machamp](/pokedex/machamp/). [Graveler](/pokedex/graveler/) → [Golem](/pokedex/golem/). [Haunter](/pokedex/haunter/) → [Gengar](/pokedex/gengar/). [Boldore](/pokedex/boldore/) → [Gigalith](/pokedex/gigalith/). [Gurdurr](/pokedex/gurdurr/) → [Conkeldurr](/pokedex/conkeldurr/).

---

## 6. Trade while holding an item

Same as #5, except the Pokémon needs to be holding a specific item during the trade. See **[Evolution Items → Trade evolutions](/evolution-items/#trade-evolutions)** for the full list.

- **Examples:** [Onix](/pokedex/onix/) + [Metal Coat](/evolution-items/#trade-while-holding-an-item) → [Steelix](/pokedex/steelix/). [Slowpoke](/pokedex/slowpoke/) + [King's Rock](/evolution-items/#trade-while-holding-an-item) → [Slowking](/pokedex/slowking/). [Porygon](/pokedex/porygon/) + [Up-Grade](/evolution-items/#trade-while-holding-an-item) → [Porygon2](/pokedex/porygon2/). [Porygon2](/pokedex/porygon2/) + [Dubious Disc](/evolution-items/#trade-while-holding-an-item) → [Porygon-Z](/pokedex/porygon-z/). [Clamperl](/pokedex/clamperl/) + [Deep Sea Tooth](/evolution-items/#trade-while-holding-an-item) → [Huntail](/pokedex/huntail/). [Clamperl](/pokedex/clamperl/) + [Deep Sea Scale](/evolution-items/#trade-while-holding-an-item) → [Gorebyss](/pokedex/gorebyss/). [Feebas](/pokedex/feebas/) + [Prism Scale](/evolution-items/#trade-while-holding-an-item) → [Milotic](/pokedex/milotic/).

---

## 7. Hold an item + level up (no trade)

Equip the item to the Pokémon's held-item slot, then level up. Often paired with a time-of-day check.

- **Examples:** [Sneasel](/pokedex/sneasel/) + [Razor Claw](/evolution-items/#held--level-up-no-trade) + level up at **night** → [Weavile](/pokedex/weavile/). [Gligar](/pokedex/gligar/) + [Razor Fang](/evolution-items/#held--level-up-no-trade) + level up at **night** → [Gliscor](/pokedex/gliscor/). [Happiny](/pokedex/happiny/) + [Oval Stone](/evolution-items/#held--level-up-no-trade) + level up during the **day** → [Chansey](/pokedex/chansey/).

---

## 8. Knowing a specific move

The Pokémon evolves the next time it levels up if it knows a particular move. Re-learn the move via the Move Tutor if needed (move tutors are in every region; see each region's PokéWorld page).

- **Examples:** [Bonsly](/pokedex/bonsly/) knowing Mimic → [Sudowoodo](/pokedex/sudowoodo/). [Mime Jr.](/pokedex/mime-jr/) knowing Mimic → [Mr. Mime](/pokedex/mr-mime/). [Aipom](/pokedex/aipom/) knowing Double Hit → [Ambipom](/pokedex/ambipom/). [Lickitung](/pokedex/lickitung/) knowing Rollout → [Lickilicky](/pokedex/lickilicky/). [Tangela](/pokedex/tangela/) knowing Ancient Power → [Tangrowth](/pokedex/tangrowth/). [Yanma](/pokedex/yanma/) knowing Ancient Power → [Yanmega](/pokedex/yanmega/). [Piloswine](/pokedex/piloswine/) knowing Ancient Power → [Mamoswine](/pokedex/mamoswine/).

---

## 9. Stat-based

A handful of Pokémon evolve into one of several forms based on which of their stats is higher when they level up.

- **[Tyrogue](/pokedex/tyrogue/)** at Lv 20:
  - Attack > Defense → **[Hitmonlee](/pokedex/hitmonlee/)**
  - Defense > Attack → **[Hitmonchan](/pokedex/hitmonchan/)**
  - Attack = Defense → **[Hitmontop](/pokedex/hitmontop/)**

---

## 10. Gender-based

Some species split into different evolutions by gender, sometimes paired with a stone.

- **[Combee](/pokedex/combee/)** *(female only)* → **[Vespiquen](/pokedex/vespiquen/)** at Lv 21. Male [Combee](/pokedex/combee/) never evolves.
- **[Burmy](/pokedex/burmy/)** → **[Wormadam](/pokedex/wormadam/)** *(female)* / **[Mothim](/pokedex/mothim/)** *(male)* at Lv 20 (Wormadam form depends on cloak).
- **[Snorunt](/pokedex/snorunt/)** *(female)* + [Dawn Stone](/evolution-items/#stones-use-on-pokémon) → **[Froslass](/pokedex/froslass/)**.
- **[Kirlia](/pokedex/kirlia/)** *(male)* + [Dawn Stone](/evolution-items/#stones-use-on-pokémon) → **[Gallade](/pokedex/gallade/)**.

---

## 11. Location-based

A few Pokémon evolve when leveled up at a specific location. On Pokéfind these locations correspond to fixed in-world regions.

- **[Magneton](/pokedex/magneton/)** at a magnetic / electric site → **[Magnezone](/pokedex/magnezone/)**.
- **[Nosepass](/pokedex/nosepass/)** at the same magnetic site → **[Probopass](/pokedex/probopass/)**.
- **[Eevee](/pokedex/eevee/)** near a Moss Rock → **[Leafeon](/pokedex/leafeon/)**. *(Pokéfind only accepts the location method — Leaf Stone does not work on Eevee.)*
- **[Eevee](/pokedex/eevee/)** near an Ice Rock → **[Glaceon](/pokedex/glaceon/)**. *(Pokéfind only accepts the location method — Ice Stone does not work on Eevee.)*

If a region's special evolution location isn't obvious, ask in chat — most regions have a single signature spot per evolution type.

---

## Things that *block* evolution

- **Holding an [Everstone](/evolution-items/).** This is the most common reason a Pokémon "won't evolve." Pop the [Everstone](/evolution-items/) out of the held-item slot and try again.
- **Cancel-evolution.** When a Pokémon is about to evolve on level-up, the prompt usually accepts B / sneak to cancel — that single attempt is skipped. The next level-up will retry.
