Pokéfind runs on **Minecraft Java Edition**. If you're on PC and have a valid Java install, this is the fastest way to join.

## Requirements

- A valid installation of **Minecraft Java Edition**.
- A **Microsoft account** properly associated with the game.
- **No modded or cracked clients** — those will not connect.
- **CurseForge cannot be used.**

Common valid installations:

- The **default Minecraft launcher** (Mojang / Microsoft).
- **Lunar Client**
- **Feather Client**

## How to join Pokéfind

1. Launch Minecraft Java Edition with your Microsoft-account-linked install.
2. From the main menu, click **Multiplayer** → **Add Server**.
3. Enter the server details:
   - **Server IP:** `play.pokefind.co`
4. Save the server, select it from the list, and click **Join Server**.

That's it — once you're in, click the **compass** in your inventory to create or join your save file. New players start in **Gen 5: the Zeinova region**; returning players land back in whichever region they were last in.
