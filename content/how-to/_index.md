A growing collection of practical setup and how-to guides for Pokéfind. Pick a guide from the sidebar or jump straight into one below.

## Available guides

- **[Join on Java](join-on-java/)** — the standard route: a valid Java Edition install with a Microsoft account, then add the server. (No modded / cracked clients; CurseForge isn't supported.)
- **[Join via Bedrock Together (Xbox / PlayStation)](bedrock-together/)** — connect a console to the Java-edition Pokéfind server through the Bedrock Together proxy.
- **[Join via Friend Accounts (Bedrock)](friend-accounts/)** — add a Pokéfind bot account as a Microsoft friend so the server appears in your Worlds tab.
- **[Clear Cached Data (Bedrock)](clear-cached-data/)** — wipe Bedrock's cached resource packs to fix invisible Pokémon, missing items, and pack-update crashes.
- **[Evolve Pokémon](evolve-pokemon/)** — every evolution method supported on Pokéfind: level-up, friendship, time of day, stones, trading, held-item-while-trading, stat-based, gender, location.

More guides will land here as topics come up. If there's a setup question that comes up often in chat or Discord, it's a candidate for this section.
